fun ObjectNode.mergeWith(rhs: ObjectNode, newObj: () -> ObjectNode, newArr: (Int) -> ArrayNode): ObjectNode

Merges JSON objects into a new JSON object containing the key/value pairs from the first overlayed with the key/value pairs from the second.

If conflicting keys are both objects or arrays, they will be merged, else the value from the second JSON object will be used.

Return

A new JSON object containing the merged input objects.

Since

v4.0.0

Parameters

rhs

Second object to merge into a copy of the receiver object.

newObj
